              <h3 class="mt-4">The Managed Desktops Division</h3>
              <p>
                I worked within Managed Desktops, a service that provides full desktop management, security, and support
                for university departments. Rather than keeping their own IT staff and infrastructure, departments can
                subscribe to this service and receive professional management of their computing environment. The
                service covers the complete desktop lifecycle:
              </p>
              <ul>
                <li>Computer procurement, setup, and provisioning</li>
                <li>Software deployment and configuration</li>
                <li>Security maintenance through SentinelOne EDR</li>
                <li>Hardware coordination, repairs, and replacements</li>
                <li>Ongoing troubleshooting and technical support</li>
                <li>Management using Microsoft Endpoint Configuration Manager</li>
              </ul>

              <h2 class="mt-5">Job Description</h2>

              <h3>Overview</h3>
              <p>
                As a Student IT Technician within the Managed Desktops division, I provided comprehensive technical
                support for university departments enrolled in the service. My daily work involved:
              </p>
              <ul>
                <li>Troubleshooting hardware and software issues across Windows and macOS systems</li>
                <li>Managing user accounts in Active Directory</li>
                <li>Deploying software through Microsoft Endpoint Configuration Manager</li>
                <li>Administering endpoint security through SentinelOne EDR</li>
                <li>Handling support tickets through BMC FootPrints</li>
                <li>Balancing desk-side visits for complex problems with remote support for quick fixes</li>
                <li>Working with technologies from network printers and peripheral devices to email systems and file
                  storage</li>
              </ul>
